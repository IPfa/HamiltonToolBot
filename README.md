# HamiltonToolBot

Телеграм бот отслеживающий срок до следующей калибровки метрологического средства измерения. Высокоточные измерительные приборы принято калибровать и проверять в определенные проможутки времени, например раз в год. HamiltonToolBot раз в месяц проверяет сколько дней осталось до следующей калибровки измерительного прибора и присылает отчет в Telegram. Бот маркирует измерительное средства, если до поверки осталось меньше девяноста дней.

# Стек Технологий
Python, API Telegram.

# Запуск
Бот можно запустить через ресурс Heroku. Список измерительных приборов можно кастомизировать.
